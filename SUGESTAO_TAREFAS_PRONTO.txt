✅ INTERFACE (Dashboard Kids)
   • Botão "💡 Sugerir Tarefa" na aba de Tarefas
   • Modal bonito e limpo
   • Formulário com 3 campos:
     - Nome da tarefa (obrigatório)
     - Descrição (opcional)
     - Pontos sugeridos (5-50, ajustável)
   • Botões + e − para pontos
   • Validação antes de enviar

✅ BANCO DE DADOS
   • Nova tabela: task_suggestions
   • Campos: título, descrição, pontos, status, etc.
   • Status: pending, approved, rejected
   • Relacionamentos com users e households

✅ API
   • POST /api/tasks/suggest - Criar sugestão
   • GET /api/tasks/suggest - Listar sugestões
   • Validação completa
   • Tratamento de erros

✅ INTEGRAÇÃO
   • Dashboard chama API ao enviar
   • Feedback visual para usuário
   • Reset do formulário após envio

PARA CRIANÇAS:
1️⃣ Entre no Dashboard Kids
2️⃣ Clique "💡 Sugerir Tarefa"
3️⃣ Preencha título, descrição e pontos
4️⃣ Envie!
5️⃣ Aguarde aprovação dos pais

PARA PAIS (Dashboard em desenvolvimento):
1️⃣ Veem lista de sugestões pendentes
2️⃣ Podem:
   ✅ Aprovar (vira tarefa normal)
   📝 Ajustar pontos
   ❌ Recusar (com motivo)
